# Drone-illustrasjoner

Legg PNG-filer med disse eksakte navnene i denne mappen (`public/drone/`).
Kildene under peker til Luftfartstilsynets sider hvor illustrasjonene finnes.
Hvis en fil mangler skjules bare bildet — resten av siden fungerer som normalt.

| Filnavn | Vises på | Hent fra |
| --- | --- | --- |
| `a1-avstand.png` | A1-regelsiden | https://luftfartstilsynet.no/droner/regelverk/apen-kategori/ (A1) |
| `a2-30m-5m.png` | A2-regelsiden | https://luftfartstilsynet.no/droner/regelverk/apen-kategori/ (A2) |
| `a3-150m.png` | A3-regelsiden | https://luftfartstilsynet.no/droner/regelverk/apen-kategori/ (A3) |
| `regel-1til1.png` | A3-regelsiden (1:1-regelen) | Søk «1:1-regelen drone» på luftfartstilsynet.no |
| `maks-hoyde-120m.png` | Alle A1/A2/A3-sidene | https://luftfartstilsynet.no/droner/regelverk/apen-kategori/ |
| `klasser-c0-c6.png` | Droneklasser-oversikten + STS | Søk «droneklasser» på luftfartstilsynet.no |

Anbefalt bredde: 1200 px, PNG, transparent eller hvit bakgrunn.
Vær oppmerksom på Luftfartstilsynets bruksvilkår ved gjenbruk av deres illustrasjoner.
